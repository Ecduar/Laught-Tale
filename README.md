"# LAUGHT-TALE"

**"Laught Tale"** es un nuestra pagina tributo a una de los mangas mas importantes de la historia.

---

## Descripción

Este es un proyecto de demostración de un sitio web temático de One Piece, construido con un enfoque Mobile-First, HTML5 semántico y CSS moderno. El proyecto incluye todos los requerimientos exigidos como un menú desplegable, un carrusel de imágenes basado en CSS y un efecto de parallax en el scroll y utiliza Webpack para optimizar y minificar los archivos HTML y CSS, preparando el sitio para producción.

---

## Características Principales

HTML5 Semántico: Uso correcto de etiquetas como <header>, <main>, <nav>, <section>, <article>, y <figure>.

Diseño Responsivo (Mobile-First): El diseño se construye desde la vista móvil y se adapta a pantallas más grandes (tablets y desktop) usando media queries.

Menú Desplegable: Menú de navegación funcional y accesible en dispositivos móviles.

Efecto Parallax: Un efecto sutil en el scroll en el banner principal, creado con JavaScript.

Carrusel de Imágenes: Un carrusel automático hecho puramente con animaciones CSS (@keyframes).

Animaciones CSS: Animaciones de entrada para las tarjetas de contenido.

Optimización de Build: Configuración de Webpack para minificar archivos HTML y CSS, y agrupar el JavaScript.

---

## Tecnologías Utilizadas

- **HTML5:** Para la estructura semántica del contenido.
- **CSS3:** Para todo el estilizado, layout, animaciones y efectos visuales.
- **JavaScript**: Para añadir interactividad al proyecto
